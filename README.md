# 🚀 Portfólio - Gustavo Sales

Este projeto é um portfólio profissional focado na apresentação de soluções reais em desenvolvimento Full Stack, integração de sistemas e automação com inteligência artificial.

## 🔥 Projeto Principal

### Plataforma de Curso com Automação Comercial

Sistema completo que integra:

- 🌐 Página de vendas (front-end)
- 🧾 Cadastro de usuários
- 🗄️ Banco de dados MySQL
- 💳 Integração com pagamento (Mercado Pago)
- 🤖 Chatbot com lógica de vendas
- 📲 Automação via WhatsApp (API Meta)
- 📊 Dashboard de acompanhamento em tempo real
- 🔁 Follow-up automático para recuperação de leads

---

## 🧠 Fluxo do Sistema

1. Usuário acessa o site
2. Realiza cadastro
3. Dados armazenados no MySQL
4. Direcionamento para pagamento
5. Se aprovado → acesso liberado
6. Se não pagar → entra em fluxo de follow-up
7. Bot entra em contato via WhatsApp
8. Dashboard acompanha tudo em tempo real

---

## ⚙️ Tecnologias utilizadas

- HTML, CSS, JavaScript
- Node.js + Express
- MySQL
- API WhatsApp (Meta)
- Mercado Pago API
- Railway (deploy e logs)

---

## 🧪 Problemas reais resolvidos

- ❌ Erro de autenticação na API do WhatsApp (OAuthException)
- 🔒 Bloqueio de requisições por CORS
- 📱 Normalização de números brasileiros
- 🔁 Controle de disparos duplicados
- ⏱️ Delay estratégico no envio de mensagens
- 📊 Organização de logs e histórico no banco

---

- Aplicação real com integração completa
- Automação comercial funcional
- Dashboard com dados reais
- Estrutura escalável
- Foco em conversão e experiência do usuário
